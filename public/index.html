What the retailer provides

  • Product catalog with ingredient data (FSSAI labels already carry it)
  • Shopper dietary profiles, one signup screen
  • The stockout event from the picker app
  • The accept or refund tap, one webhook

Scaling path

  • This demo: in-memory graph, 195 SKUs, instant
  • Production: identical schema on Neo4j, millions of SKUs
  • One LLM call per stockout, about one second
  • Stateless, horizontally scalable
